# Aegis Agentic Product OS — Project PRD (Frontend)

The frontend is a premium chat-driven dashboard for Aegis.

## Key Frontend Features
- **Workspace Navigation:** Perplexity-style glassmorphic chat (Home).
- **Mission Control:** Dashboard for health scores and tracking active bets.
- **Directions:** Manual or clustered strategic direction declaration.
- **Inbox:** Workflow for approving or rejecting proposed interventions (HITL).
- **Activity:** Historical log of agent debates and risk detections.

## Interactions
- **Connect Workspace:** User connects Linear.
- **Declare Bet:** Fill modal with hypothesis and target segment.
- **Scan Linear:** Trigger a pipeline scan from the dashboard.
- **Review Intervention:** Examine rationale and approve/reject.
- **Chat:** Converse with Aegis to describe new strategies and get risk summaries.

## Technical Details
- **Next.js 14+:** App Router.
- **React 19:** Latest features.
- **CopilotKit:** Integration with Backend ADK for real-time state sync.
- **Tailwind 4:** Modern styling with custom glass effects.
- **Lucide Icons:** Unified iconography.

## Design Aesthetic
- **Theme:** Premium Light Glassmorphism.
- **Palette:** HSL tailored colors (slate-900, indigo, violet).
- **Interactions:** Hover effects, smooth transitions, and real-time polling updates.
